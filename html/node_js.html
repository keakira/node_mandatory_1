Node.js
Node.js er en server og en standalone interpretor til JavaScript.

Node.js interpretoren er lavet til at køre på en computer serverside for således at gøre det muligt at lave backend applicationer i JavaScript. Node.js interpretoren er baseret på en V8 interpretor som er indbygget i Crome.

Node.js er en web application server som kører JavaScript på samme måde som Tomcat er en server som kører Java.

Med Node.js følger NPM package manager.
