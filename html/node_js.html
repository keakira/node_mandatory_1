Node.js

Node.js er en server og en standalone interpretor til JavaScript.

Node.js interpretoren er lavet til at køre på en computer serverside for således at gøre det muligt at lave backend applicationer i JavaScript. Node.js interpretoren er baseret på V8 interpretor'en som er indbygget i Crome. Node interpreter'en tiljøjer server-relateret funktionalitet som ikke er tilgængelig i en browser, fordi browseren skal kunne køre "untrusted" kode, som derfor skal sandboxes.

Af Node-ekskulsiv funktionalitet kan nævnes, at åbne porte eller indlæse filer fra disk uden file-selektor dialoger. Til gengæld præsenterer V8-interpreteren et mere specialiseret miljø hvor det kan forventes at have ting som et window og en DOM stillet til rådighed, mens dette ikke nødvendigvis er tilfældet med Node-interpreteren, fordi denne stiller et mere generisk miljø til rådighed

Det vil sige at Node.js er en web application server som kører JavaScript på samme måde som Tomcat er en server som kører Java.

Med Node.js følger NPM package manager (svarende til Maven til Java).

Nodemon

Nodemon er et "hot-reload"-værktøj som kan installeres globalt på computeren. Der er tale om et værktøj som kan bruges ved developement (ikke produktion) til at køre programmet. Nodemon er et værktøj som køres på sammen måde som node køres. Det tager sig af at køre node og automatisk genstarte den, hvis filer i projekt-mappen ændres.
Kør JavaScript-filer med Node.js
1) I terminal, navigér ind i projekt-mappen
2) Giv javascript-filen som argument til node 
node myscript.js
Installer og brug Nodemon
Installer nodemon globalt på computeren
1) I terminal:
 
npm install -g nodemon
Start hot-reload med nodemon på projektet
2) I terminal:
 
nodemon myscript.js
